**What to build:** A deployable Next.js 16 (App Router) web app in a new top-level `web/`, themed to the dark-terminal design via shadcn CSS-variable tokens, that builds with Turbopack and previews/deploys to Cloudflare via the OpenNext adapter — showing a minimal dark-themed page with one shadcn component.
