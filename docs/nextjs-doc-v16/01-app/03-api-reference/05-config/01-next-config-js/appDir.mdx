> **Good to know**: This option is **no longer** needed as of Next.js 13.4. The App Router is now stable.

The App Router ([`app` directory](../../../index.mdx)) enables support for [layouts](../../03-file-conventions/layout.mdx), [Server Components](../../../01-getting-started/05-server-and-client-components.mdx), [streaming](../../03-file-conventions/loading.mdx), and [colocated data fetching](../../../01-getting-started/06-fetching-data.mdx).

Using the `app` directory will automatically enable [React Strict Mode](https://react.dev/reference/react/StrictMode). Learn how to [incrementally adopt `app`](../../../02-guides/migrating/app-router-migration.mdx#migrating-from-pages-to-app).
