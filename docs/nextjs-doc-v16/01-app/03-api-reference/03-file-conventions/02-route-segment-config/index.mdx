title: Route Segment Config

The Route Segment Config options allow you to configure the behavior of a [Page](../page.mdx), [Layout](../layout.mdx), or [Route Handler](../route.mdx) by directly exporting the following variables:

| Option                                                                                             | Type                                                 | Default                    |
| -------------------------------------------------------------------------------------------------- | ---------------------------------------------------- | -------------------------- |
| [`dynamicParams`](dynamicParams.mdx)     | `boolean`                                            | `true`                     |
| [`runtime`](runtime.mdx)                 | `'nodejs' \| 'edge'`                                 | `'nodejs'`                 |
| [`preferredRegion`](preferredRegion.mdx) | `'auto' \| 'global' \| 'home' \| string \| string[]` | `'auto'`                   |
| [`maxDuration`](maxDuration.mdx)         | `number`                                             | Set by deployment platform |
