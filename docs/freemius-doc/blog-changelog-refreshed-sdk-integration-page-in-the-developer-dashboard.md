[Changelog](https://freemius.com/changelog/) / Refreshed SDK Integration page in the Developer Dashboard

Our [Developer Dashboard](https://dashboard.freemius.com/) has a dedicated page where you can get instructions and configure different parameters for our official [WordPress SDK](https://github.com/Freemius/wordpress-sdk). We noticed, that over time the page got bloated with much information.

[![](https://freemius.com/blog/wp-content/uploads/2024/08/sdk-integration-freemius-developer-dashboard-1024x766.png)](https://freemius.com/blog/wp-content/uploads/2024/08/sdk-integration-freemius-developer-dashboard.png)

To simplify it for first-time users, we have refreshed the page. It now holds the necessary information ordered first and links to our [documentation](https://freemius.com/help/documentation/) where it dives deep into the details.
