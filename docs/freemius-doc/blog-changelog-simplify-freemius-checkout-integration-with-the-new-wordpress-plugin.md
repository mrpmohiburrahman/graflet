[Changelog](https://freemius.com/changelog/) / Simplify Freemius Checkout Integration with the New WordPress Plugin

We’re excited to share the official release of the [**Freemius for WordPress**](help-documentation-wordpress-freemius-for-wordpress.md) plugin — a community-built initiative by **Xaver**, the creator of [**Mailster**](https://mailster.co/). Developed independently and supported by Freemius, the plugin makes it incredibly easy to integrate the Freemius Checkout and other components into your WordPress site. Watch the short video below to explore its features.

If you’re selling your WordPress product with Freemius, then besides the built-in WP SDK Checkout, we’ve always recommended embedding a pricing page on your own marketing website using the Freemius Checkout JS SDK. However, this required writing and maintaining the integration code manually.

With the **Freemius for WordPress plugin**, you no longer have to. If your theme supports blocks, you can turn any button into a Freemius Checkout button and configure all available options right from the editor. You can also integrate Freemius Checkout with any pricing table or layout pattern that comes with your theme.

To make this even easier for our makers, we’ve added a new step in the **Setup Checklist** inside the **Developer Dashboard** that explains the plugin’s role and how it fits into your setup.

[![Access Freemius for WP Plugin from the Setup Checklist](https://freemius.com/blog/wp-content/uploads/2025/11/freemius-for-wp-setup-checklist-1024x343.png)](https://freemius.com/blog/wp-content/uploads/2025/11/freemius-for-wp-setup-checklist.png)

Additionally, you’ll now find instructions for using the plugin inside the **Get Checkout Code** dialog accessible from the **Plans** page.

[![Freemius for WordPress plugin shown inside the Get Checkout Code](https://freemius.com/blog/wp-content/uploads/2025/11/freemius-wp-checkout-code-1024x555.png)](https://freemius.com/blog/wp-content/uploads/2025/11/freemius-wp-checkout-code.png)

You can learn more in our [documentation](help-documentation-wordpress-freemius-for-wordpress.md).

Even if your product itself isn’t built on WordPress, as long as your marketing site runs on WordPress, you can use this plugin to easily integrate Freemius Checkout and other components.

We’re just getting started with this plugin. In upcoming releases, we plan to introduce more features such as [embedding the Customer Portal](https://github.com/Freemius/freemius-users-dashboard/), [providing SSO login options](help-documentation-users-account-management-sso-single-sign-on-wordpress.md), and unifying all these capabilities into a single, robust plugin for WordPress makers.
