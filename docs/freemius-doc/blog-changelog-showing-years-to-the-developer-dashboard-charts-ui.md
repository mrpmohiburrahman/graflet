[Changelog](https://freemius.com/changelog/) / Showing years to the Developer Dashboard Charts UI

We’ve done a minor enhancement to the charts UI in the Developer Dashboard.

[![Charts hover date range on Freemius Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2024/04/showing-year-freemius-developer-dashboard-chart-1024x491.png)](https://freemius.com/blog/wp-content/uploads/2024/04/showing-year-freemius-developer-dashboard-chart.png)

While hovering on a point, if the selected date range spans over different years, the UI will show the full year of the given node. This helps quickly understand which year the point falls on.
