[Changelog](https://freemius.com/changelog/) / PayPal Flow Fix for WP Admin In-Dashboard Upgrades

The [WP SDK in-dashboard upgrade](https://freemius.com/help/documentation/wordpress-sdk/features/wp-admin-in-dashboard-upgrading/) flow is designed to let users upgrade without leaving the product experience. In some rare edge cases, however, the PayPal payment flow could fail during that process.

[![PayPal upgrade flow from the WordPress admin](https://freemius.com/blog/wp-content/uploads/2026/07/freemius-wp-checkout-1024x522.png)](https://freemius.com/blog/wp-content/uploads/2026/07/freemius-wp-checkout.png)

We identified the root cause and deployed a fix, making the PayPal upgrade path more reliable for users purchasing directly from your [WordPress](help-documentation-wordpress.md) product’s admin.
