[Changelog](https://freemius.com/changelog/) / Data export now available for Subscriptions

Following up on our [data liberation project](blog-changelog-enhancements-to-the-users-export-from-the-developer-dashboard.md), we are glad to announce the immediate availability of subscription exports from the [Developer Dashboard](https://dashboard.freemius.com/#!/live/new-product/).

Previously we worked on improving the [User Export](blog-changelog-enhancements-to-the-users-export-from-the-developer-dashboard.md) and also introduced the [Payment Export](blog-changelog-introducing-new-feature-to-export-payments.md).

[![](https://freemius.com/blog/wp-content/uploads/2024/09/freemius-subscription-export-1024x478.png)](https://freemius.com/blog/wp-content/uploads/2024/09/freemius-subscription-export.png)

With this week’s release exporting Subscriptions is now possible. Go to the “Subscription” page under your Developer Dashboard and click the “Download” button to start.

We are not done with the data liberation project yet. Please stay tuned for more updates.
