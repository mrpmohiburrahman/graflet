[Changelog](https://freemius.com/changelog/) / Store Settings Page Now Organized with Tabs

This week we are rolling out a useful update to the Store Settings page in the [Freemius Developer Dashboard](https://dashboard.freemius.com). Previously, all store configurations were displayed on a single page, which made it difficult to quickly find the relevant settings. Many makers also shared similar feedback with us.

To improve the UX, we are now grouping related settings into separate tabs.

[![Store page in the Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2026/03/dashboard-store-page-1024x697.png)](https://freemius.com/blog/wp-content/uploads/2026/03/dashboard-store-page.png)

For example, the basic configuration is now grouped under the “Information” tab. If you’re looking to embed the [Customer Portal](help-documentation-users-account-management.md), you can now find it under the “Customer Portal” tab.

[![Customer Portal config under stores](https://freemius.com/blog/wp-content/uploads/2026/03/dashboard-store-customer-portal-1024x913.png)](https://freemius.com/blog/wp-content/uploads/2026/03/dashboard-store-customer-portal.png)

Additionally, we have fixed a few minor UI glitches across the Developer Dashboard. We hope this change makes it easier for you to navigate and manage your store settings.
