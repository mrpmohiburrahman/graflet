We hope everyone is staying safe and healthy in these wild times 😷 The team has been hard at work (albeit in quarantine) with some awesome new features 🚀 around refunds, filters in the Developer Dashboard, making affiliate links more accessible, and gifts and surprises for Freemius partners! Let’s jump right in and see what new monetization features are available for your business 👉

### Partial Refunds

We’re excited to share that Freemius now fully supports *Partial Refunds*! We’ve had tons of requests for this one because use-cases for partial refunds arise pretty often, like in cases where a coupon code wasn’t applied properly or your customer requests a prorated refund.

While partial refunds can be useful for a lot of reasons, don’t forget to set expectations properly upfront with the [right refund policy](blog-refund-policy-wordpress-plugins-themes.md) to minimize the amount of refunds you issue.

![Freemius Partial Refunds - Selecting Refund Amount](https://freemius.com/blog/wp-content/uploads/2020/04/freemius-partial-refunds-selecting-refund-amount.png)

### Late Refunds

Remember how you needed to contact us every time you had to issue a refund for a payment older than 30 days? No more! Developers no longer have to contact support for processing late refunds that are outside the 30-day refund window 🎉

PayPal payments can now be refunded up to 180 days after the transaction was processed, and credit card payments will now be refundable for 365 days:

![Freemius Late Refund Notice Regarding Fees](https://freemius.com/blog/wp-content/uploads/2020/04/freemius-late-refund-notice-regarding-fees.png)

This is part of an overall focus for 2020, in which we are going to prioritize features that allow you to manage capabilities that were previously only available for our internal team.

Keep in mind that due to financial reasons (bookkeeping & sales taxes), refunds processed after the 30-day refund period will not be eligible for a refund of the Freemius revenue share.

## Payments

After [recently adding support for GBP](blog-opt-in-permissions-privacy-transparency-payout-notifications.md), we created a currency filter for anyone selling with multiple currencies, so now you can now easily filter by USD, EUR, and GBP directly in the Payments area.

![Freemius Developer Dashboard - Filtering Payments by Currency](https://freemius.com/blog/wp-content/uploads/2020/04/freemius-developer-dashboard-filtering-payments-by-currency.gif)

If you’ve been testing how the [Checkout or Buy Button](help-documentation-selling-with-freemius-freemius-checkout-buy-button.md) works, you can now hide test payments so they won’t get mixed in with your live customer payments and cramp your style 🕺

![Freemius Developer Dashboard - Hide Test Payments](https://freemius.com/blog/wp-content/uploads/2020/04/freemius-developer-dashboard-hide-test-payments.png)

You can now easily set a license quota to any limit instead of having to click each time you want to add a license. This is a simple UI enhancement, but it sure can save your fingers from all that extra clicking if someone has a 50-site license 🤣

![Freemius Developer Dashboard - Setting License Quota Limit](https://freemius.com/blog/wp-content/uploads/2020/04/freemius-developer-dashboard-setting-license-quota-limit.gif)

## Affiliate Platform

If you’re using the [*Affiliate Platform*](help-documentation-affiliate-platform-affiliate-program-activation.md), we added affiliate links directly on the affiliate profile sidebar in the *Developer Dashboard*, so in the case that an affiliate contacts you asking for their unique links, you can easily locate and copy them:

![Freemius Affiliate Links in Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2020/04/freemius-affiliate-links-in-developer-dashboard.png)

Affiliates already have these details emailed to them when the relationship starts and can also access their links via the *Affiliate Portal*. But, we found that in many cases developers need to easily access an affiliate’s links so we decided to make those available to both sides.

If you’re having trouble finding or [onboarding](help-documentation-affiliate-platform-onboarding-affiliate-marketers.md) affiliates, here’s the [The Ultimate Guide To An Affiliate Program](blog-affiliate-program-wordpress-plugins-themes.md) for plugins and themes.
