[Changelog](https://freemius.com/changelog/) / Introducing Seamless License Upgrades from Products to Bundles

Just as we gear up for our annual Black Friday / Cyber Monday promotion, we’re rolling out a highly [requested improvement](https://freemius.nolt.io/53). Your customers can now upgrade from individual products to [bundles](help-documentation-wordpress-selling-bundles-and-memberships.md) directly from the [Checkout](help-documentation-checkout.md)—smoothly, reliably, and without needing any manual guidance.

In many stores, customers often start with a single product and later discover the value of upgrading to a bundle. Until now this process wasn’t as straightforward as it should have been.

With this release, if a customer already owns a license for one of the products included in a bundle, they can simply use the same license key during Checkout to upgrade. They’ll automatically receive [prorated discounts](https://freemius.com/help/documentation/checkout/proration/), keep the same license key across all bundled products, and retain all existing [website activations.](help-documentation-wordpress-license-utilization.md)

Below is how to make sure your products and bundles are set up correctly.

### How to Set Up a Bundle

Let’s say you have a plugin called “My Awesome Plugin,” and you want to allow an upgrade to a bundle named “My Awesome Bundle,” which might include additional products like a theme or widgets.

To enable upgrades, all you need to do is ensure that the bundle contains a plan referencing the product.

You can learn more in our [documentation](help-documentation-wordpress-selling-bundles-and-memberships.md#the-process), but the screenshot below illustrates how the bundle plan already includes the product.

[![Set up bundle plan](https://freemius.com/blog/wp-content/uploads/2025/11/setup-bundle-plan-1024x679.png)](https://freemius.com/blog/wp-content/uploads/2025/11/setup-bundle-plan.png)

That’s all the configuration required. If you already have existing bundles and product plans in your store, you likely already have this set up.

### How Customers Upgrade to the Bundle

To start the upgrade flow, ask your customers to open the Checkout of the bundle—either through a [hosted link](https://freemius.com/help/documentation/checkout/hosted-checkout/) or [pricing buttons on your website](https://freemius.com/help/documentation/checkout/freemius-checkout-buy-button/).

Next, they simply enter the license key they already own for the individual product.

[![](https://freemius.com/blog/wp-content/uploads/2025/11/enter-license-key-for-upgrade-1024x964.png)](https://freemius.com/blog/wp-content/uploads/2025/11/enter-license-key-for-upgrade.png)

Checkout will automatically recognize the license as part of the bundle’s offering and apply the appropriate prorated discount.

[![](https://freemius.com/blog/wp-content/uploads/2025/11/license-key-bundle-proration-1024x559.png)](https://freemius.com/blog/wp-content/uploads/2025/11/license-key-bundle-proration.png)

The customer completes the Checkout, and the upgrade is finished.

### What Happens to the Upgraded License

1. The license becomes a bundle license, unlocking all products included in that bundle plan.
2. The license adopts the bundle’s plan and pricing—allowing you to craft attractive upsell incentives by offering both bundle and plan upgrades in one step.
3. All existing activations remain intact. Customers won’t need to re-activate anything.
