[Changelog](https://freemius.com/changelog/) / Added 2FA recovery and backup code support for the Developer Dashboard

Following our latest feature release of the [2FA security enhancement](blog-changelog-new-security-feature-two-factor-authentication-in-the-developer-dashboard.md), we’re pleased to announce the availability of recovery or backup code support. Please see the authentication flow below:

### Enabling 2FA with a backup code

When you enable 2FA using the same method as before, a backup code will be generated automatically for you.

[![](https://freemius.com/blog/wp-content/uploads/2023/12/Saving-2FA-backup-code-1.png)](https://freemius.com/blog/wp-content/uploads/2023/12/Saving-2FA-backup-code-1.png)

Please click on the copy button and store the backup code somewhere safe.

### Regenerating the backup code

In case you’ve lost your backup code or enabled 2FA before we rolled out the feature, you can still re-generate it as long as you have your 2FA authenticator device with you.

[![](https://freemius.com/blog/wp-content/uploads/2023/12/2FA-Regenerate-code.png)](https://freemius.com/blog/wp-content/uploads/2023/12/2FA-Regenerate-code.png)

From the Profile page, simply click on the ‘generate a backup code’ link and follow the on-screen instructions.

[![](https://freemius.com/blog/wp-content/uploads/2023/12/2FA-enter-code-for-regenerating-backup.png)](https://freemius.com/blog/wp-content/uploads/2023/12/2FA-enter-code-for-regenerating-backup.png)

It will ask for the authenticator code.

[![](https://freemius.com/blog/wp-content/uploads/2023/12/2FA-save-backup-code.png)](https://freemius.com/blog/wp-content/uploads/2023/12/2FA-save-backup-code.png)

Upon entering a valid code, you will be given a new backup code. Please note that the old backup code will stop working as soon as you generate a new one.
