[Changelog](https://freemius.com/changelog/) / PAYOUTS log in the Affiliates Section

Before now, there was no way to check which affiliates had their balances paid once an affiliate had been marked as ‘Paid Out’.

We’ve implemented a feature in the PAYOUTS tab of the Affiliate page that allows you to filter the list of affiliates by their balance status.[![](https://freemius.com/blog/wp-content/uploads/2022/10/freemius-affiliate-balance-filter.png)](https://freemius.com/blog/wp-content/uploads/2022/10/freemius-affiliate-balance-filter.png)

With the help of the filter, you can list affiliates whom you’ve set as ‘Paid Out’ and view information about when the balances were marked as such.

[![](https://freemius.com/blog/wp-content/uploads/2022/10/freemius-affiliate-balance-paid-out.png)](https://freemius.com/blog/wp-content/uploads/2022/10/freemius-affiliate-balance-paid-out.png)

Please login to the [Developer Dashboard](https://dashboard.freemius.com/) (also available in the [Beta Developer Dashboard](https://dashboard-beta.freemius.com/)) to start using this function. If you want to get started with the Freemius Affiliate Platform, you can find all the essential information [here](help-documentation-affiliate-platform.md).
