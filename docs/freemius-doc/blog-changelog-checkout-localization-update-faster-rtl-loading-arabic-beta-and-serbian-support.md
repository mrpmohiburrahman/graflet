[Changelog](https://freemius.com/changelog/) / Checkout Localization Update: Faster RTL Loading, Arabic Beta, and Serbian Support

This week, we released the following improvements to our Checkout, aimed at improving [localization](help-documentation-checkout-features-local-languages-currencies.md).

## RTL Improvements

RTL languages and page styling now load faster, and the layout shifts immediately when a buyer selects any RTL language.

[![RTL language Freemius Checkout](https://freemius.com/blog/wp-content/uploads/2026/06/rtl-language-support-freemius-checkout-1024x557.png)](https://freemius.com/blog/wp-content/uploads/2026/06/rtl-language-support-freemius-checkout.png)

We have also released Arabic language support in beta.

[![Arabic language support in Freemius Checkout](https://freemius.com/blog/wp-content/uploads/2026/06/arabic-language-freemius-checkout-1024x564.png)](https://freemius.com/blog/wp-content/uploads/2026/06/arabic-language-freemius-checkout.png)

The translation currently targets **Modern Standard Arabic**. To use it, pass `ar` as the [`language`](help-documentation-checkout-integration-freemius-checkout-buy-button.md#language%20%7C%20locale) value. We will roll it out in production mode as we receive more feedback from our makers. If you would like to help, please get in touch with our support team from the [Developer Dashboard](https://dashboard.freemius.com).

## Serbian Language Now Available

[![Serbian Language in Freemius Checkout](https://freemius.com/blog/wp-content/uploads/2026/06/serbian-language-freemius-checkout-1024x556.png)](https://freemius.com/blog/wp-content/uploads/2026/06/serbian-language-freemius-checkout.png)

Serbian language Srpski is now generally available in our Checkout. You can pass `sr` as the `language` value, or use `auto` and your buyers from Serbia will automatically see the Checkout in their local language.
