[Changelog](https://freemius.com/changelog/) / Improvements to the events log table

Following our push to improve the table UX of the Developer Dashboard, we’ve made subtle improvements to the events log table.

[![](https://freemius.com/blog/wp-content/uploads/2024/01/freemius-developer-dashboard-events-log-table-ux-1024x306.png)](https://freemius.com/blog/wp-content/uploads/2024/01/freemius-developer-dashboard-events-log-table-ux.png)

1. You will see that hovering on the **Type** will give you the full text.
2. You can use the copy button to quickly copy the event type for your custom code.
3. The **Created on** column has been made consistent with [other tables](blog-changelog-table-ui-ux-improvements-for-the-developer-dashboard.md).
4. The **Trigger** will also show the type and have a copy button for copying the user or developer ID.

You will find these improvements wherever the events log table is visible, for example:

[![](https://freemius.com/blog/wp-content/uploads/2024/01/freemius-developer-dahboard-events-and-webhooks-1024x360.png)](https://freemius.com/blog/wp-content/uploads/2024/01/freemius-developer-dahboard-events-and-webhooks.png)

- The **Events Log** page.
- **Events & Webhooks** under the **Sites** and **Users**, etc.
