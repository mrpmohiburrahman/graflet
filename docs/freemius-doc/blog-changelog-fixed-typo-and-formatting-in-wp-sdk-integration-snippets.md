[Changelog](https://freemius.com/changelog/) / Fixed Typo and Formatting in WP SDK Integration Snippets

We noticed a minor issue in the [WP SDK Integration code](help-documentation-wordpress-sdk-integrating-freemius-sdk.md) generated by our [Developer Dashboard](https://dashboard.freemius.com). These integration snippets are simple copy-paste blocks that you can use to integrate your [WordPress Plugin or Theme](help-documentation-wordpress.md) with the Freemius WP SDK in minutes.

The issue involved a small typo and a formatting glitch.

[![WordPress SDK Integration code snippet](https://freemius.com/blog/wp-content/uploads/2025/10/wp-sdk-integration-code-1024x610.png)](https://freemius.com/blog/wp-content/uploads/2025/10/wp-sdk-integration-code.png)

We’ve now fixed the typo and adjusted the formatting to ensure the snippet is error-free and follows WordPress’ standard code styling.
