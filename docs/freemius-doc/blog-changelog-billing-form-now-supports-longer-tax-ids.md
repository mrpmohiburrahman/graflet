[Changelog](https://freemius.com/changelog/) / Billing Form Now Supports Longer Tax IDs

We have updated our [billing form](help-documentation-selling-with-freemius-your-earnings.md#set-up-billing-information-to-receive-invoices) to accept Tax IDs of up to 18 characters. Previously, it was limited to 16 characters, which did not work well for some regions, such as China.

[![Billing Tax ID update](https://freemius.com/blog/wp-content/uploads/2026/05/billing-tax-id-update-1024x465.png)](https://freemius.com/blog/wp-content/uploads/2026/05/billing-tax-id-update.png)

As Freemius continues expanding support for makers from different regions, small updates like this help make the platform more flexible for local billing and compliance requirements.

If you have any other issues, please feel free to contact our support.
