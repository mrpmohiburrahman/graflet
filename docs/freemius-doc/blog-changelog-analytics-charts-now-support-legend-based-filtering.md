[Changelog](https://freemius.com/changelog/) / Analytics Charts Now Support Legend-Based Filtering

This week we’re rolling out a useful improvement to our [analytics](help-documentation-analytics.md). You can now click any chart legend item to filter the chart by the selected items only.

[![Freemius charts filtering](https://freemius.com/blog/wp-content/uploads/2026/04/filtering-country-chart-1024x484.png)](https://freemius.com/blog/wp-content/uploads/2026/04/filtering-country-chart.png)

This makes it easier to inspect your data in more detail. For example, if you want to view only a single country or just a few countries in the [payments breakdown chart](help-documentation-analytics-sales.md#visualized-sales-data), you can now select those countries directly from the legend and the chart will render only the relevant data.

[![Freemius Payments Charts](https://freemius.com/blog/wp-content/uploads/2026/04/filtering-payment-chart-1024x486.png)](https://freemius.com/blog/wp-content/uploads/2026/04/filtering-payment-chart.png)

We hope you enjoy this new feature. If you have more suggestions, please let us know in our [feature board](https://freemius.nolt.io/).
