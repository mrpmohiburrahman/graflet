[Changelog](https://freemius.com/changelog/) / Refinements and Layout Fixes in the Customer Portal

This week we are rolling out a set of fixes addressing UI glitches in our Customer Portal.

[![Freemius Customer Portal](https://freemius.com/blog/wp-content/uploads/2026/02/customer-portal-layout-1024x388.png)](https://freemius.com/blog/wp-content/uploads/2026/02/customer-portal-layout.png)

1. We identified some elements that were still using legacy hard-coded colors, which could cause inconsistencies across [themes](help-documentation-users-account-management-applying-css-customization.md). These have now been updated to use the correct design tokens.
2. We reviewed and refined all tables to ensure consistent spacing and alignment, even for products with shorter titles.
3. We also resolved a few minor layout glitches to improve overall visual stability and polish.

Read our [documentation](help-documentation-users-account-management.md) if you want to learn more about our Customer Portal.
