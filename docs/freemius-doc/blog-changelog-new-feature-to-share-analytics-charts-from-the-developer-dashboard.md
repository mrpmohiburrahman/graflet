[Changelog](https://freemius.com/changelog/) / New feature to share analytics charts from the Developer Dashboard

Following the trend of #buildinpublic, we’ve created a cool feature for you in the Developer Dashboard. You can now share any chart that you see in the Developer Dashboard.

[![Share charts in the Freemius Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2023/10/freemius-developer-dashboard-share-chart.jpg)](https://freemius.com/blog/wp-content/uploads/2023/10/freemius-developer-dashboard-share-chart.jpg)

### Sharing Features

[![Different chart sharing features](https://freemius.com/blog/wp-content/uploads/2023/10/freemius-developer-dashboard-sharing-features.jpg)](https://freemius.com/blog/wp-content/uploads/2023/10/freemius-developer-dashboard-sharing-features.jpg)

- You can switch between different background gradients and colors.
- You can choose to hide numbers from the chart.
- You can toggle between light and dark themes for the chart, regardless of the theme of the Developer Dashboard.

Here’s a sample:

[![Sample chart screenshot - Freemius](https://freemius.com/blog/wp-content/uploads/2023/10/freemius-net-revenue-nov-2022-oct-2023.png)](https://freemius.com/blog/wp-content/uploads/2023/10/freemius-net-revenue-nov-2022-oct-2023.png)

We hope you will enjoy this feature. Please let us know your feedback.
