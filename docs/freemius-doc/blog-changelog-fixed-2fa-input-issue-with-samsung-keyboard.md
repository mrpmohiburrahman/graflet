[Changelog](https://freemius.com/changelog/) / Fixed 2FA input issue with Samsung Keyboard

Freemius supports securing your account with [two-factor authentication](help-documentation-security-two-factor-authentication-2fa.md). We recently discovered that the 2FA input was not accepting pasted codes from Samsung Keyboard’s clipboard bubble.

[![](https://freemius.com/blog/wp-content/uploads/2025/03/freemius-2fa-ui-login-1024x919.png)](https://freemius.com/blog/wp-content/uploads/2025/03/freemius-2fa-ui-login.png)

This issue has now been fixed, enabling smoother 2FA code pasting from the keyboard toolbar.
