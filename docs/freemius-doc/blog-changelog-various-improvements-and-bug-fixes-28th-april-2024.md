[Changelog](https://freemius.com/changelog/) / Various improvements and bug fixes – 28th April 2024

### Improvements to the Readme.txt missing notice in the Deployment page

We noticed that the message we were showing the [Deployment](https://freemius.com/help/documentation/selling-with-freemius/deployment/) page, had some visualization glitch.

[![](https://freemius.com/blog/wp-content/uploads/2024/04/freemius-deployment-notice-missing-readme-1024x349.png)](https://freemius.com/blog/wp-content/uploads/2024/04/freemius-deployment-notice-missing-readme.png)

We have fixed the glitches and made improvements to the message itself.

### Fixed broken `geo.json` endpoint

We noticed a regression where the endpoint `https://checkout.freemius.com/geo.json` got broken. It is a feature Freemius provides using which you can determine the location to [dynamically set currency](help-documentation-selling-with-freemius-freemius-checkout-buy-button.md) on your pricing page. We have fixed the regression.
