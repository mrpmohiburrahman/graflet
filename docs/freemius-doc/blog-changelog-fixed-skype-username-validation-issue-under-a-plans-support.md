[Changelog](https://freemius.com/changelog/) / Fixed Skype username validation issue under a plan’s support

We noticed that our system was rejecting any Stripe usernames containing the `:` character. In this iteration, we have fixed this issue.

[![Fixed skype username validation issue](https://freemius.com/blog/wp-content/uploads/2024/04/better-skype-username-validation-freemius-developer-dashboard-1024x496.png)](https://freemius.com/blog/wp-content/uploads/2024/04/better-skype-username-validation-freemius-developer-dashboard.png)

You can add a Skype username under the Developer Dashboard’s Support section of a plan.
