[Changelog](https://freemius.com/changelog/) / Dashboard Responsiveness and Notification Fixes

This week we are rolling out several Quality of Life improvements to our Developer Dashboard.

### Charts and Controls are better on all screen sizes

We improved the responsiveness of [charts](help-documentation-analytics-insights-insights-dashboard.md) and their controls to display better across different screen sizes.

[![Freemius Charts Responsiveness Improvements](https://freemius.com/blog/wp-content/uploads/2025/09/charts-improvements-1024x789.png)](https://freemius.com/blog/wp-content/uploads/2025/09/charts-improvements.png)

They now adapt more intelligently to the available space, ensuring a smoother experience whether on desktop or smaller screens.

### Previous and Next Button on all Sidebar Panes

During the [rebranding](blog-freemius-rebrand-website-developer-dashboard-redesign.md) we introduced next and previous buttons for some sidebar panes that accompany tables. These provide a great UX by letting you quickly navigate through different items without going back to the table view.

[![Sidepane navigation in Freemius Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2025/09/next-prev-button-developer-dashboard-side-panes-1024x343.png)](https://freemius.com/blog/wp-content/uploads/2025/09/next-prev-button-developer-dashboard-side-panes.png)

This week we’ve consolidated the feature by bringing the same navigation to **all** sidebar panes, making item browsing more consistent across the Dashboard.

### Fixed Notification UI for New Developers

Freemius allows you to [customize the notifications](help-documentation-marketing-automation-transactional-emails.md#email-notification-customization) you receive as a seller. We noticed a regression where, for some new makers, accessing the customization page caused it to break in certain cases.

[![Notification Config from the Freemius Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2025/09/freemius-maker-notification-config-1024x463.png)](https://freemius.com/blog/wp-content/uploads/2025/09/freemius-maker-notification-config.png)

This edge case has now been fixed, ensuring a smooth notification setup experience for all makers.
