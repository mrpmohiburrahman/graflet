Our [JS SDK](help-documentation-saas-sdk-js-sdk.md) is framework-agnostic and can be easily integrated into your Fastify application.

While we don’t yet provide an official Fastify adapter, you can use the [basic integration steps](help-documentation-saas-sdk-js-sdk-integration.md) and adjust them to Fastify’s plugin and route system.

We may consider offering a dedicated Fastify adapter or guide based on community demand. If this interests you, please upvote the feature request on our [public suggestion board](https://freemius.nolt.io/).
