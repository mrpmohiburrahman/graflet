Freemius currently offers two ways to distribute software updates to your users via the Customer Portal and the WordPress SDK Integration.

### WordPress SDK Integration[​](#wordpress-sdk-integration "Direct link to WordPress SDK Integration")

For WordPress products, Freemius provides the [WordPress SDK](help-documentation-wordpress-sdk.md), which when integrated will make the update process feel more native within the WordPress admin dashboard.

This enables you to effectively manage the [updates distribution](help-documentation-wordpress-software-updates-distribution.md) for your **premium plugins, add-ons, and themes**.

### Download from Customer Portal[​](#download-from-customer-portal "Direct link to Download from Customer Portal")

The [Freemius Customer Portal](help-documentation-users-account-management.md) empowers your users to easily access and download the latest versions of your software products. Users only need to log in to their accounts and download available zip files via the [**Downloads**](help-documentation-users-account-management-downloads.md) page.

This helps distribute updates for your WordPress products as well as templates and widgets.
