[Changelog](https://freemius.com/changelog/) / Extra Security for “Login as User” in the Developer Dashboard

The Freemius Developer Dashboard allows you to view detailed information about your [users](help-documentation-analytics-user-data.md#your-users). One of the most useful capabilities is being able to log in as a user to the [Customer Portal](help-documentation-users-account-management.md), making it easier to reproduce, inspect, and debug issues firsthand.

[![Login as user Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2026/01/login-as-user-1024x344.png)](https://freemius.com/blog/wp-content/uploads/2026/01/login-as-user.png)

With this week’s deployment, we’ve strengthened the security around this flow. When logging in to the Customer Portal as a user, you will now be prompted to confirm your Developer Dashboard password.

[![Required developer password for logging in to the Customer Portal](https://freemius.com/blog/wp-content/uploads/2026/01/developer-customer-portal-login-form-1024x828.png)](https://freemius.com/blog/wp-content/uploads/2026/01/developer-customer-portal-login-form.png)

This additional confirmation helps ensure the feature remains secure, even in scenarios where a device may be left unattended or otherwise compromised.
