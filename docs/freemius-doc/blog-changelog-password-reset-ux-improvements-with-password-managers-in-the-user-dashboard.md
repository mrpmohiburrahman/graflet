[Changelog](https://freemius.com/changelog/) / Password reset UX improvements with password managers in the Customer Portal

We learned that some password managers like LastPass, were not providing a good UX when our buyers tried to reset their passwords.

[![Freemius password reset email](https://freemius.com/blog/wp-content/uploads/2024/05/freemius-reset-password-email-1024x587.png)](https://freemius.com/blog/wp-content/uploads/2024/05/freemius-reset-password-email.png)The issues happened especially when a buyer has multiple accounts with Freemius.

In some cases, it would force fill in incorrect email addresses during the reset, while in some cases it would not prompt you to save the password against the right account. We have pushed a fix to improve this UX.

[![](https://freemius.com/blog/wp-content/uploads/2024/05/freemius-user-dashboard-password-reset-form-1024x924.png)](https://freemius.com/blog/wp-content/uploads/2024/05/freemius-user-dashboard-password-reset-form.png)

Now LastPass will prompt for a new auto-generated password in the reset form.

[![](https://freemius.com/blog/wp-content/uploads/2024/05/freemius-user-dashboard-password-update-lastpass-1024x690.png)](https://freemius.com/blog/wp-content/uploads/2024/05/freemius-user-dashboard-password-update-lastpass.png)

Once saved, LastPass will ask for a password update against the right account.
