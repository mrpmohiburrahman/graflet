[Changelog](https://freemius.com/changelog/) / Customer Portal Modal Positioning Fixed in Embed Mode

A maker reported an issue with our [Customer Portal](help-documentation-users-account-management.md) where, in [embed mode](help-documentation-users-account-management-embedding-customer-portal.md#embedding-the-customer-portal-in-your-store), some modal dialogs could go outside the viewport. This happened only when the portal was embedded inside a fixed-height container.

[![Showing a Customer Portal modal dialog staying within the visible embedded viewport](https://freemius.com/blog/wp-content/uploads/2026/06/customer-portal-dialog-positioning-1024x337.png)](https://freemius.com/blog/wp-content/uploads/2026/06/customer-portal-dialog-positioning.png)

This has now been fixed, so modal dialogs should remain properly visible within the embedded portal experience.
