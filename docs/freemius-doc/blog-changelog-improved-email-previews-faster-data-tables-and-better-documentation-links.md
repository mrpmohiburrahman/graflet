[Changelog](https://freemius.com/changelog/) / Improved Email Previews, Faster Data Tables, And Better Documentation Links

This week, we focused on improving the DX around the [Developer Dashboard](https://dashboard.freemius.com).

## Improved Email Customization UI

We noticed some issues in the preview UI of [Email Customization](help-documentation-marketing-automation-email-content-customization.md), which sometimes confused our makers.

[![Showing the improved Email Customization preview UI](https://freemius.com/blog/wp-content/uploads/2026/06/email-customization-ui-improvement-1024x635.png)](https://freemius.com/blog/wp-content/uploads/2026/06/email-customization-ui-improvement.png)

We pushed an update to make the preview section more dynamic and better reflect the type and configuration of the product.

## Improved Data Table Loading Performance

Last month, we rolled out an update to [improve the performance of several data tables](blog-changelog-developer-dashboard-gets-faster-data-loading-and-safer-price-deletion.md). However, after some production testing, we found that there was still room for improvement.

[![Faster data table loading in the Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2026/06/freemius-dashboard-data-tables-1024x680.png)](https://freemius.com/blog/wp-content/uploads/2026/06/freemius-dashboard-data-tables.png)

This week, we’re rolling out another set of changes to improve loading performance even further. We hope this improves your experience while browsing [subscriptions, payments, or licenses](help-documentation-getting-started-explore-the-developer-dashboard.md) in the [Developer Dashboard](https://dashboard.freemius.com).

## Better Documentation Links

Freemius documentation goes deep for both [WordPress products](help-documentation-wordpress.md) and other software, such as [SaaS and apps](help-documentation-saas.md). We understand that the documentation can feel a bit overwhelming for first-time makers.

[![contextual documentation links in the Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2026/06/freemius-documentation-1024x743.png)](https://freemius.com/blog/wp-content/uploads/2026/06/freemius-documentation.png)

To make it easier for our makers to find the right information, we updated several documentation links in the Dashboard to pre-select the type of software you’re selling with Freemius. We hope this makes the documentation feel more relevant and helps you get to the right guidance faster.
