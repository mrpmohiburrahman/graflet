[Changelog](https://freemius.com/changelog/) / Fix: Inconsistent Relative Date Display in Developer Dashboard

We noticed that the relative date shown in some parts of our [Developer Dashboard](https://dashboard.freemius.com) had an inconsistency, especially when the value fell within a specific range.

[![Relative date in the Developer Dashboard UI](https://freemius.com/blog/wp-content/uploads/2025/10/relative-date-time-fix-1024x656.png)](https://freemius.com/blog/wp-content/uploads/2025/10/relative-date-time-fix.png)

We’ve rolled out a fix to address this issue. Relative date and time displays are now consistent across the entire Dashboard.
