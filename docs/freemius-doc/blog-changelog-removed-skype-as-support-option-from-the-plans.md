[Changelog](https://freemius.com/changelog/) / Removed Skype as Support Option from the Plans

Historically, Freemius allowed adding Skype as one of the [support channels configured for a plan](help-documentation-getting-started-setup-product-pricing-plans-refunds.md#configure_a_plan_s_support_channels). This information would appear in various places such as post-purchase emails, helping your buyers easily reach out for [support](help-documentation-selling-with-freemius-email-settings.md).

[![Freemius plan configuration for adding support channels.](https://freemius.com/blog/wp-content/uploads/2025/05/freemius-plan-support-config-1024x422.png)](https://freemius.com/blog/wp-content/uploads/2025/05/freemius-plan-support-config.png)

With Skype being gradually phased out, we have now removed the functionality and the associated UI from our system. For those who had Skype set up as a support option, it has been automatically removed and will no longer appear in any relevant areas of the platform.
