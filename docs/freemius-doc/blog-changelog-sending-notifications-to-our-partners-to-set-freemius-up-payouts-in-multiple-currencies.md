[Changelog](https://freemius.com/changelog/) / Sending notifications to our partners to set Freemius up payouts in multiple currencies

Freemius supports selling in multiple [currencies](help-documentation-selling-with-freemius-multi-currency.md). As a partner, you can go to the PLANS sections of the Developer Dashboard to set up pricing in currencies other than the default USD.

Selling in multiple currencies does require setting up different payouts. However, we won’t ask you to do that unless you have pending balances in other currencies.

To automate the process, we’ll send an email notification to you if you reach a predetermined payout balance in another currency for which you have not yet set up a payout method.

### Setting up payouts in multiple currencies

Please follow our official guide on [Getting Paid](help-documentation-selling-with-freemius-your-earnings.md).

[![Setting up multi-currency payout method Freemius](https://freemius.com/blog/wp-content/uploads/2022/11/freemius-payout-method-different-currency.png)](https://freemius.com/blog/wp-content/uploads/2022/11/freemius-payout-method-different-currency.png)

When you start configuring your Payout Method(s), you’ll see an option to set different payout methods for all configured currencies.
