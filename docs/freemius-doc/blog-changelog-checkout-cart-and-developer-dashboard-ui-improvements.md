[Changelog](https://freemius.com/changelog/) / Checkout Cart and Developer Dashboard UI improvements

### Cart and coupon bug fix

We identified an issue with the checkout’s cart system where entering a coupon would break the cart updation process. We’ve pushed a fix for this.

### 2FA UI/UX improvements

[![Freemius Login - 2FA UI now supports copy/paste](https://freemius.com/blog/wp-content/uploads/2023/12/freemius-2fa-copy-paste.gif)](https://freemius.com/blog/wp-content/uploads/2023/12/freemius-2fa-copy-paste.gif)

Following maker feedback, we’ve added **copy/paste** support in the 2FA input.

### Fixing UI regression on the Payout method’s Account type

We noticed a UI regression where we were mistakenly showing ‘Verified/Unverified’ in the ‘Account type’ dropdown when the payout method was set as ‘Bank Wire’.

[![Freemius Payout Account Type](https://freemius.com/blog/wp-content/uploads/2023/12/freemius-payout-account-type.png)](https://freemius.com/blog/wp-content/uploads/2023/12/freemius-payout-account-type.png)

We’ve pushed a fix to the UI regression – no updates from your end are necessary.
