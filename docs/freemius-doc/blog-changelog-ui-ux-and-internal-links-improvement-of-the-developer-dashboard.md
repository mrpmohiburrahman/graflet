[Changelog](https://freemius.com/changelog/) / UI/UX and Internal Links Improvement of the Developer Dashboard

This week, we’re releasing a few minor but helpful UI/UX and internal link improvements to the [Developer Dashboard](https://dashboard.freemius.com/).

1. The **API documentation** links now point to the correct [updated documentation site](blog-changelog-launch-of-our-api-documentation.md).
2. Under **Plans → Code/Link**, we’ve added direct links to our [documentation](help-documentation-selling-with-freemius-freemius-checkout-buy-button.md). This should help new makers better understand how to implement the [overlay code](help-documentation-selling-with-freemius-freemius-checkout-buy-button.md) and [no-code](help-documentation-selling-with-freemius-hosted-checkout.md) links of our Checkout.
3. We also fixed a UI issue with **popovers**, where multiple popovers could stay open at the same time. This has now been resolved.
