Freemius integrates with [Help Scout](help-documentation-integrations-help-scout.md) and [MailChimp](help-documentation-integrations-mailchimp-integration.md) out of the box. However, you can also leverage our [webhooks](help-documentation-saas-events-webhooks.md) and REST API to integrate with any 3rd-party tool, automating workflows using the data Freemius gathers and segments about your software users and customers.

For instance, you can use webhooks to push user data to your favorite CRM and mailing platforms like ConvertKit or Customer.io.
