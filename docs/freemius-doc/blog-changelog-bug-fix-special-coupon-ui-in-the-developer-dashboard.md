[Changelog](https://freemius.com/changelog/) / Bug Fix: Special Coupon UI in the Developer Dashboard

Freemius allows you to set up [special coupons](help-documentation-marketing-automation-special-coupons-discounts.md) to boost your sales and marketing efforts. Simply head over to **Coupons → Special Coupons** in the Developer Dashboard to get started.

[![Freemius special coupons UI](https://freemius.com/blog/wp-content/uploads/2025/04/freemius-special-coupon-1024x809.png)](https://freemius.com/blog/wp-content/uploads/2025/04/freemius-special-coupon.png)

We recently discovered a UI bug where clicking the **Remove** button to reset a special coupon would break the interface, requiring a page reload to recover. This issue has now been fixed.
