[Changelog](https://freemius.com/changelog/) / Easier API authentication with Bearer Auth protocol

To push Freemius for SaaS products, we have released yet another new feature. Now it is easier to communicate with our API using Bearer Authentication.

[![](https://freemius.com/blog/wp-content/uploads/2024/09/freemius-api-auth-bearer-token-1024x644.png)](https://freemius.com/blog/wp-content/uploads/2024/09/freemius-api-auth-bearer-token.png)

Go to “Settings” &gt; “API Token” of any product and you will see the token that you can use as [bearer auth](https://datatracker.ietf.org/doc/html/rfc6750#section-6.1.1). We have also included a few examples to get you started.

Please note that the API Token works only for product-scoped endpoints for now. We are also working on a new [API documentation website](https://freemius.com/help/documentation/api/) using OpenAPI schema to ease the consumption of different endpoints. Please stay tuned for more updates.
