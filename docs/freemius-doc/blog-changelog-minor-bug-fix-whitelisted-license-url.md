[Changelog](https://freemius.com/changelog/) / Minor Bug Fix: Whitelisted License URL

Freemius includes a feature that allows customers or buyers to [whitelist a license](help-documentation-users-account-management-license-security.md#url_whitelisting_aka_license_firewall), enabling activation only on selected sites.

[![](https://freemius.com/blog/wp-content/uploads/2024/12/freemius-customer-portal-license-security-1024x599.png)](https://freemius.com/blog/wp-content/uploads/2024/12/freemius-customer-portal-license-security.png)

We identified a UI regression where the requested URL occasionally failed to appear in the whitelist. The root cause has been addressed, and a fix has been deployed.
