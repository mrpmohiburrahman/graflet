## Mobile UX improvements

We noticed that our checkout’s footer was being cut off on some mobile devices.

[![](https://freemius.com/blog/wp-content/uploads/2024/03/freemius-checkout-on-mobile-device-483x1024.png)](https://freemius.com/blog/wp-content/uploads/2024/03/freemius-checkout-on-mobile-device.png)

In this iteration, we fixed this regression. Additionally, we also improved the native scrolling experience on mobile browsers. The mobile menu bar will now automatically be hidden when scrolling down, giving more real estate to the Checkout app.

## Fixed a typo

We’d been using the term “Annually” in the billing cycle UI for a long time. While this was not technically incorrect, the term ‘Annual’ fits better (kudos to our Support Lead, [Leo,](https://twitter.com/vienlhanz26) and Digital Copywriter, [Robert](https://twitter.com/RobertNolte10)).

[![](https://freemius.com/blog/wp-content/uploads/2024/03/freemius-checkout-billing-cycle-ui-annual-term-1024x490.png)](https://freemius.com/blog/wp-content/uploads/2024/03/freemius-checkout-billing-cycle-ui-annual-term.png)

In this update, we changed the term.
