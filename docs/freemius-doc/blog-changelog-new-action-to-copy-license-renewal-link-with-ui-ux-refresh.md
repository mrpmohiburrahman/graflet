[Changelog](https://freemius.com/changelog/) / New action to copy license renewal link with UI/UX refresh

We made some noticeable UI/UX overhauls to the Licenses page in your [Developer Dashboard](https://dashboard.freemius.com/). We wanted to add a new action item in the table and while doing so, we noticed that over the time, the table UI has gotten bloated with many actions.

So we took the opportunity to make some refreshing changes.

## Refreshed Licenses table with consolidated actions

We replaced some old designs and components inside the licenses table.

[![Freemius Developer Dashboard Licenses table page](https://freemius.com/blog/wp-content/uploads/2024/07/better-table-layout-freemius-licenses-page-1024x583.png)](https://freemius.com/blog/wp-content/uploads/2024/07/better-table-layout-freemius-licenses-page.png)

For example, we removed some space between the columns and used a new component to display the license key with the copy and reveal button.

[![Grouped actions button inside Freemius Developer Dashboard's License page](https://freemius.com/blog/wp-content/uploads/2024/07/grouped-action-button-freemius-licenses-page-1024x598.png)](https://freemius.com/blog/wp-content/uploads/2024/07/grouped-action-button-freemius-licenses-page.png)

We grouped all available actions, which are non-frequent, at the end of the column inside a dropdown menu.

We hope this change will help you make the most use of the table, by showing the most important things at first.

## New action to copy renewal link

Sometimes customers ask our makers to share some links from where they can renew expired licenses or update billing methods. We do provide a [Customer Portal](https://freemius.com/help/documentation/users-account-management/users-dashboard/) from where your customers can self-serve. But sometimes sharing a direct link helps in quicker conversion.

[![Copy renewal link action inside Licenses page of Freemius Developer Dashboard](https://freemius.com/blog/wp-content/uploads/2024/07/copy-renewal-link-freemius-licenses-page-1024x993.png)](https://freemius.com/blog/wp-content/uploads/2024/07/copy-renewal-link-freemius-licenses-page.png)

To help you quickly generate such links, we have added a new action item beside every license row. Simply click on the “**Copy Renewal Link**” button and share the link with your customer. This will open the checkout with the license key prefilled from where they can update the renewal mechanism.
