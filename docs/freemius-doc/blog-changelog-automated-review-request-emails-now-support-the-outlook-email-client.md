[Changelog](https://freemius.com/changelog/) / Automated review request emails now support the Outlook email client

Freemius sends [review request emails](help-documentation-marketing-automation-reviews.md) to your buyers as part of [marketing automation](help-documentation-marketing-automation.md). We learned that the review request emails were not working in some email clients like Outlook.

[![Freemius request to review email form](https://freemius.com/blog/wp-content/uploads/2024/05/freemius-review-email-form-877x1024.png)](https://freemius.com/blog/wp-content/uploads/2024/05/freemius-review-email-form.png)

We have deployed a fix for the same.
