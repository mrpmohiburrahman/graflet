[Changelog](https://freemius.com/changelog/) / Customer Portal UI & Customization Fixes

This week we’re rolling out a few minor quality-of-life improvements to our [Customer Portal](help-documentation-users-account-management.md).

- We refined the error UI for better readability, and fixed an issue where some links appeared with an incorrect color.
- We fixed a bug where [custom CSS](help-documentation-users-account-management-applying-css-customization.md) wasn’t updating reliably after makers changed their styling.
