[Changelog](https://freemius.com/changelog/) / Clearer ZIP Deployment Errors and Better Self-Help Guidance

We noticed that some makers were running into [deployment](help-documentation-wordpress-deployment-process.md) issues because of how their ZIP files were created. While our system was able to detect these issues, the UI did not clearly communicate what went wrong or how to fix it. To make troubleshooting easier and provide better self-help guidance, we revamped the UI.

[![Deployment error UI](https://freemius.com/blog/wp-content/uploads/2026/03/deployment-error-ui-1024x344.png)](https://freemius.com/blog/wp-content/uploads/2026/03/deployment-error-ui.png)

Known issues are now identified more clearly, and each one links to the relevant documentation for additional context. We also updated our [documentation](help-documentation-release-management-deployment.md) with OS-specific guides for creating ZIP files, along with additional troubleshooting resources.
