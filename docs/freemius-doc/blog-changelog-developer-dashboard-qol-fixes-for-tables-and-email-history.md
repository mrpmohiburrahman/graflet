[Changelog](https://freemius.com/changelog/) / Developer Dashboard QoL Fixes for Tables and Email History

We have deployed the following quality-of-life updates to the [Developer Dashboard](https://dashboard.freemius.com).

We identified cases where the horizontal scrollbar was not visible, especially on pages with long tables. This made it harder to inspect wider datasets. The issue has now been fixed.

[![Freemius Dashboard Horizontal Scroll](https://freemius.com/blog/wp-content/uploads/2026/03/freemius-long-table-horizontal-scroll-1024x589.png)](https://freemius.com/blog/wp-content/uploads/2026/03/freemius-long-table-horizontal-scroll.png)

We also noticed a few UI glitches and alignment issues in the recently released [Email History feature](blog-changelog-see-all-buyer-emails-sent-by-freemius-in-one-place.md). Those have now been fixed as well.

[![Freemius Email History](https://freemius.com/blog/wp-content/uploads/2026/03/freemius-email-history-1024x504.png)](https://freemius.com/blog/wp-content/uploads/2026/03/freemius-email-history.png)
