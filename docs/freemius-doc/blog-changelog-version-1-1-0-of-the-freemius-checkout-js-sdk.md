[Changelog](https://freemius.com/changelog/) / Version 1.1.0 of the Freemius Checkout JS SDK

We’re excited to announce the release of version **1.1.0** of the Freemius [Checkout JS SDK](https://freemius.com/blog/changelog_category/js-sdk/)! This update introduces a backward-compatible enhancement, allowing you to use **`product_id`** instead of **`plugin_id`** (because we support [various kinds of products](freemius-for-saas.md), not only plugins).

[![](https://freemius.com/blog/wp-content/uploads/2024/12/checkout-js-sdk-type-1024x879.png)](https://freemius.com/blog/wp-content/uploads/2024/12/checkout-js-sdk-type.png)

We also improved the type system to provide better TypeScript intellisense.

The change is also reflected in our Developer Dashboard.

[![](https://freemius.com/blog/wp-content/uploads/2024/12/checkout-js-sdk-code-1024x728.png)](https://freemius.com/blog/wp-content/uploads/2024/12/checkout-js-sdk-code.png)

Simply go to the Plans page and click on the **Get Checkout Code** button to get started.

If you’re using the [CDN version](https://github.com/Freemius/freemius-checkout-js/?tab=readme-ov-file#using-hosted-cdn) of the Checkout JS, no actions are necessary from your end. If you’re using [NPM or other package manager](https://github.com/Freemius/freemius-checkout-js/?tab=readme-ov-file#usage-guide), be sure to update to the latest version.
