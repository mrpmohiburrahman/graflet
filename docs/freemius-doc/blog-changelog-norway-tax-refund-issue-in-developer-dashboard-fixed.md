[Changelog](https://freemius.com/changelog/) / Norway Tax Refund Issue in Developer Dashboard Fixed

Freemius supports [tax refunds](help-documentation-selling-with-freemius-refund-payment.md#tax-refunds) against a valid tax ID. For example, if a customer forgot to enter their VAT number during checkout, you can issue the refund from the Developer Dashboard. The associated subscription will then be updated and the tax amount will be [exempted](help-documentation-selling-with-freemius-refund-payment.md#how-to-refund-vat) using the reverse charge mechanism.

[![Norway tax refund](https://freemius.com/blog/wp-content/uploads/2026/03/norway-payment-tax-refund-1024x631.png)](https://freemius.com/blog/wp-content/uploads/2026/03/norway-payment-tax-refund.png)

We identified an issue in the Developer Dashboard where the tax refund option for [Norway](blog-changelog-vat-collection-and-remittance-now-supported-for-norway.md) was not working properly. This has now been fixed.
