[Changelog](https://freemius.com/changelog/) / Developer Dashboard improvements – Security email + better UX for countries picker

### Added a new security email to the profile page

Following our [last security disclosure](blog-freemius-wordpress-sdk-security-vulnerability.md), we received feedback from our partners that they would like the security emails to be sent to a separate address. In response, we have added a new section in the “[Profile](https://dashboard.freemius.com/#!/profile/)” page of the [Developer Dashboard](https://dashboard.freemius.com/).

[![security email freemius developer dashboard profile page](https://freemius.com/blog/wp-content/uploads/2023/07/security-email-freemius-developer-dashboard-profile.jpg)](https://freemius.com/blog/wp-content/uploads/2023/07/security-email-freemius-developer-dashboard-profile.jpg)

From here, you can add an email address where Freemius will send you security-related notifications. Please note that Freemius will send the notification to both the main email as well as the “Security email”.

### Better UX for country pickers

Following yet another bit of feedback, we’ve improved the areas in the Developer Dashboard where we present a dropdown for selecting a country. For example:

- Billing Information
- Gifts & Surprises
- Payout Bank Account

etc.

[![](https://freemius.com/blog/wp-content/uploads/2023/07/searchable-country-picker-freemius-developer-dashboard.jpg)](https://freemius.com/blog/wp-content/uploads/2023/07/searchable-country-picker-freemius-developer-dashboard.jpg)

As you can see, you can now easily search for the country you intend to pick.
