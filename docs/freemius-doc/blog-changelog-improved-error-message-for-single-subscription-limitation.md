[Changelog](https://freemius.com/changelog/) / Improved Error Message for Single Subscription Limitation

Previously, we released a feature that allows you to [enforce a single-subscription model for your SaaS](blog-changelog-new-saas-feature-restrict-one-subscription-per-user.md).

When this feature is enabled, our Checkout shows an error message if an already subscribed user attempts to subscribe again.

[![Improved error UI in the Freemius Checkout for Single Subscription model, linking to the currently active subscription management page](https://freemius.com/blog/wp-content/uploads/2025/05/single-subscription-ui-improvement-1024x556.png)](https://freemius.com/blog/wp-content/uploads/2025/05/single-subscription-ui-improvement.png)

With this release, we’ve enhanced the UI by automatically including a link that directs users to upgrade their subscription. The link leads to the Freemius-managed [**Customer Portal**](help-documentation-users-account-management.md), offering an out-of-the-box experience for your buyers to manage their purchases and licenses.
