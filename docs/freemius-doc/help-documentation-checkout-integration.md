This guide provides step-by-step instructions on how to seamlessly integrate the Freemius checkout into your application, ensuring a smooth and efficient payment process for your customers.

Learn how to add the checkout to your website, using the:

- [Overlay Checkout JavaScript API](help-documentation-checkout-integration-overlay-checkout.md): To create a buy button that triggers the Freemius overlay checkout.
- [Freemius Hosted Checkout](help-documentation-checkout-integration-hosted-checkout.md): Share a direct link to a secure, hosted checkout page with the ability to redirect customers after purchase.
- [Checkout API](help-documentation-checkout-integration-freemius-checkout-buy-button.md): To customize the checkout experience and handle various events using callbacks.
- [Renewal & Updating links](help-documentation-checkout-integration-generate-renewal-payment-method-update-link.md): To generate links for customers to renew subscriptions or update their payment methods.
