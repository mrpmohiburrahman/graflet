[Changelog](https://freemius.com/changelog/) / New Freemius Checkout JS SDK Out of Beta

We recently [announced](blog-changelog-new-javascript-sdk-for-freemius-checkout-calling-for-testers.md) the beta availability of the new version of the [Freemius Checkout JS SDK](help-documentation-selling-with-freemius-freemius-checkout-buy-button.md). Today, we are excited to share that the library is officially out of beta. A huge thank you to everyone who helped us test it in real-world scenarios!

[![](https://freemius.com/blog/wp-content/uploads/2024/11/checkout-js-sdk-code-1024x725.png)](https://freemius.com/blog/wp-content/uploads/2024/11/checkout-js-sdk-code.png)

Our Developer Dashboard and website documentation have been updated to reflect these changes. Here’s a brief recap of what the new JS SDK offers:

1. **No dependencies**: The new SDK does not require jQuery or other dependencies.
2. **Improved performance**: It loads faster than the legacy integration library.
3. **Package manager support**: The SDK can be installed via package managers like NPM (refer to our [GitHub repository](https://github.com/Freemius/freemius-checkout-js?tab=readme-ov-file#usage-guide) for details).

We will soon retire our old library and replace it with the [adapter](https://github.com/Freemius/freemius-checkout-js/?tab=readme-ov-file#migration-adapter-not-recommended) the new library provides. However we urge that you [update to the new library](https://github.com/Freemius/freemius-checkout-js/?tab=readme-ov-file#migration-guide) as soon as possible to take full advantage of the modular system.
