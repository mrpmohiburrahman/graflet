[Changelog](https://freemius.com/changelog/) / New feature to duplicate coupon

We are glad to announce a new Developer Dashboard feature to duplicate coupons easily.

[![](https://freemius.com/blog/wp-content/uploads/2024/04/coupon-duplicate-feature-freemius-developer-dashboard-1024x680.png)](https://freemius.com/blog/wp-content/uploads/2024/04/coupon-duplicate-feature-freemius-developer-dashboard.png)

You can click on the new “Duplicate” button to get started. It will create a new coupon copying all its settings and attributes. The newly created coupon code will become, `OLDCODE_COPY` which you can change from the edit screen.

[![](https://freemius.com/blog/wp-content/uploads/2024/04/duplicated-coupon-freemius-developer-dashboard-1024x562.png)](https://freemius.com/blog/wp-content/uploads/2024/04/duplicated-coupon-freemius-developer-dashboard.png)

Please note that if you’re copying a product-level coupon from the Store view, the duplicated coupon by default will only apply to the product itself.

[![](https://freemius.com/blog/wp-content/uploads/2024/04/coupon-configuration-freemius-1024x629.png)](https://freemius.com/blog/wp-content/uploads/2024/04/coupon-configuration-freemius.png)

You can edit the coupon to make the necessary adjustments, for example, you can activate it for more products, plans, and billing cycles etc.
