[Changelog](https://freemius.com/changelog/) / Minor Bug Fixes Across the System – Reverse Invoices & UI

This week we are releasing several small but important bug fixes across our system:

- **Reverse Invoices**: We fixed an issue where the system would occasionally fail to generate the [reverse invoice](blog-freemius-release-notes-april-2025.md#no_more_manual_invoicing_for_payouts_seriously).
- **MailChimp Integration UI**: A minor UI issue was identified in the [MailChimp integration form](help-documentation-integrations-mailchimp-integration.md) within the [Developer Dashboard](https://dashboard.freemius.com/). This has been fixed.
- **[Customer Portal](help-documentation-users-account-management.md) Updates**:
  
  - The portal was still displaying our old logo — this has now been updated.
  - We also improved the copy around the login form to make it clearer and more user-friendly.
