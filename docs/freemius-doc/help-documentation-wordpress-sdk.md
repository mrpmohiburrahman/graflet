# Exploring the Freemius WordPress SDK

The WordPress SDK enables plugin and theme developers to integrate their products with Freemius and work with WordPress sites seamlessly.

The SDK provides a powerful set of features that allow you to manage your product's licensing, account management, and in-dashboard upgrade/pricing pages, among other functionalities. The SDK ensures that only authorized users can access premium features while free users can upgrade.

This tool is core to Freemius, enabling a natural workflow of development and release cycles for both free and premium versions of your product.

This guide provides context about the Freemius WordPress SDK integrated into your WordPress plugin, theme, or addon. It explains the [integration process](https://freemius.com/help/help/documentation/wordpress-sdk/integration/.md), [various features](https://freemius.com/help/help/documentation/wordpress-sdk/features/.md), [customizations](https://freemius.com/help/help/documentation/wordpress-sdk/customization/.md), and [testing & debugging tools](https://freemius.com/help/help/documentation/wordpress-sdk/testing/.md) to enable you to identify and resolve issues efficiently during development.

* [License activation screen](https://freemius.com/help/help/documentation/wordpress-sdk/features/opt-in-screen/.md#license-activation-screen)
* [Licensing handling](https://freemius.com/help/help/documentation/wordpress-sdk/integration/software-licensing/.md)
