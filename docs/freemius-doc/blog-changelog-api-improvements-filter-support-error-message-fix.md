[Changelog](https://freemius.com/changelog/) / API Improvements: Filter Support & Error Message Fix

This week we’re deploying a couple of minor but meaningful bug fixes in the [API](https://docs.freemius.com/api):

- The [List all Payments](https://docs.freemius.com/api/payments/list) endpoint now properly supports the `filter` parameter. Many makers rely on filtered payment lookups for dashboards and internal tooling, so ensuring this parameter behaves consistently was important.
- We fixed a small typo in an error message generated by the [Update License](https://docs.freemius.com/api/licenses/update) endpoint to improve clarity and reduce confusion during integrations.
