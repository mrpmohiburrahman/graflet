[Changelog](https://freemius.com/changelog/) / Complete Developer Dashboard reskinning – with dark mode

This weekend, it’s our pleasure to announce a wholly reskinned and modernized Developer Dashboard.

[![](https://freemius.com/blog/wp-content/uploads/2023/02/reskinning-light-theme-1024x905.jpg)](https://freemius.com/blog/wp-content/uploads/2023/02/reskinning-light-theme-scaled.jpg)

[![](https://freemius.com/blog/wp-content/uploads/2023/02/reskinning-dark-theme-1024x905.jpg)](https://freemius.com/blog/wp-content/uploads/2023/02/reskinning-dark-theme-scaled.jpg)

The entirety of the application has been reskinned. When you log in for the first time, the system will try to detect your color preference and automatically set the light or the dark theme. You can click on the sun/moon icon beside the Freemius logo to toggle the theme as you please.
